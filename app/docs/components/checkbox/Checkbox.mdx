## Table of Contents

The Checkbox Component is a user interface element that allows users to toggle between two states, typically representing selection or deselection. It consists of a small box that can be checked (selected) or unchecked (deselected). Checkboxes are commonly used in forms, lists, and interactive interfaces to enable users to make multiple selections or indicate their preferences.

## Default Checkbox

The Default Checkbox is a basic user interface element that represents a standard checkbox control. It allows users to toggle between two states: checked and unchecked.

## Checkbox With Level

The Checkbox With Level component extends the standard checkbox functionality by introducing hierarchical levels. To add level in checkbox you need to add this props `level="Checkbox"`.
