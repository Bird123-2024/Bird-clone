## Table of Contents

The Preloader component in the Keep Design System provides a visual indication of an ongoing process or loading state. With various options for type and device, you can choose the appropriate preloader style to enhance the user experience on both web and mobile platforms.

## Default Skeleton

The Default Skeleton component provides a visual placeholder that mimics the appearance of loading content. It presents a line of skeleton components and serves as a foundation for different skeleton layouts.
