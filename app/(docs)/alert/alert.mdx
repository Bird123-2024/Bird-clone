## Default Alert

The default alert is a simple and clean design without any extra visual elements. It's perfect for straightforward notifications that don't require any specific styling.

## Alert With Dissmiss Icon

The dismiss icon allows users to close an alert when needed. To add the dismiss icon to the alert, you should pass a handler function to the onDismiss prop.

</CodePreview>

## Rounded Alert

If you want a softer and more friendly look, the rounded alert is a great choice. By setting the `rounded` prop to `true`, the alert will have rounded corners, giving it a gentle and approachable appearance.

<CodePreview code={AlertWithRoundedCode}>
  <AlertWithRounded />
</CodePreview>

## Alert With Border

Adding a border to an alert can make it stand out and draw attention. Use the `withBorder` prop set to `true` to give the alert a distinct border.

<CodePreview code={AlertWithBorderCode}>
  <AlertWithBorder />
</CodePreview>

## Alert With Border Accent

For a touch of elegance, the border-accented alert is a fantastic option. Set the `withBorderAccent` prop to `true`, and by default, the border accent will appear on the `left` side of the alert. For a right-side accent, use `withBorderAccentPosition="right"`. You can even customize the `top` and `bottom` positions.

<CodePreview code={AlertWithBorderAccentCode}>
  <AlertWithBorderAccent />
</CodePreview>

## Alert With Color Variant

Colorful alerts can help differentiate various types of messages. By default, the alert color is `info`, but you can choose from a range of color options. Simply set the `color` prop to `success`, `warning`, `error`, `gray`, or `info` to match the alert with your design theme.

<CodePreview code={AlertWithColorVariantCode}>
  <AlertWithColorVariant />
</CodePreview>
